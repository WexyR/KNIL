Entrer dans le dossier "Knil Dungeon"
Lancer le jeu via la commande "python main.py" dans une console
Pour commencer une nouvelle partie, selectionner "nouvelle partie", appuyer sur "e"
Pour reprendre la partie en cours, selectionner "Reprendre la partie", appuyer sur "e"
Pour quitter, selectionner "Quitter", appuyer sur "e"

Pour jouer:
z, q, s, d: déplacer le joueur
i: ouvrir l'inventaire
j: ouvrir le journal des quêtes
e: intéraction / validation
